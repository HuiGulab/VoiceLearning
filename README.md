## 学习计划

前期以深度学习和音频的基础理论知识为主，搭配paddlespeech开源程序库进行实践练习；学习过程中了解体验其他企业推出的语音智能交互产品。之后逐渐开始阅读相关学术论文，明确研究方向和论文
